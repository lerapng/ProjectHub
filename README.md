# ProjectHub - Система управління проєктами

Веб-додаток для організації завдань та нотаток, розроблений у межах навчальної практики.

## Технологічний стек
- **Frontend:** React + TypeScript (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Backend-as-a-Service:** Supabase (PostgreSQL, Auth, RLS)
- **Deployment:** Vercel (CI/CD інтеграція з GitHub)

## Функціонал
- **Автентифікація:** Реєстрація та вхід користувачів через Supabase Auth.
- **Управління проєктами:** Створення, редагування та видалення робочих просторів.
- **Канбан-дошка:** Система завдань із різними пріоритетами та статусами (To Do, In Progress, Done).
- **Нотатки:** Вбудований редактор для швидких записів усередині кожного проєкту.
- **Безпека (RLS):** Кожен користувач має доступ лише до власних даних завдяки політикам Row Level Security на рівні бази даних.

## Як запустити локально
1. Клонувати репозиторій.
2. Встановити залежності: `npm install`.
3. Запустити в режимі розробки: `npm run dev`.
